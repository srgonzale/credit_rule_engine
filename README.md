# Credit Rule Engine

## How to Run

To run the main application, execute the following command from the root directory:

```bash
python3 -m credit_rule_engine.main
```

## Project Structure

- `credit_rule_engine/`: Main package
  - `main.py`: Entry point
  - `models.py`: Data models
  - `rules.py`: Rule definitions
  - `engine.py`: Rule engine logic

## Running Tests

To run the unit tests, execute the following command from the root directory:

```bash
python3 run_tests.py
```
